Features

慣れた操作に、
タブの自由を。

Everything familiar. Now with better flow.

Windows の作法を置き換えず、複数フォルダーを行き来する日常だけをもっと軽やかにします。

＋
Vertical Tabs

迷わず使える垂直タブ

新規タブ、固定、複製、復元、ドラッグ並べ替え、中クリックで閉じる。フォルダーをタブバーへドロップすれば新しいタブで開きます。固定タブと幅は次回起動時に復元されます。

Ctrl+TCtrl+WCtrl+TabPinDrop to open
⋯
Real Windows Shell

本物のシェルメニュー

コピー、削除、プロパティ、Open with、拡張機能が追加した項目まで、Windows 標準の右クリックメニューがそのまま動きます。「新規作成」メニューも Explorer と同じ内容です。

↔
Explorer Compatible

コピーも DnD も相互互換

Explorer と Kiriha の間で切り取り・コピー・貼り付け、ドラッグ＆ドロップが自然につながります。削除はごみ箱へ、進捗も Windows 標準ダイアログで表示。

⌕
Search

探す操作をすぐ近くに

検索ボックスに入力するだけで、現在のフォルダーの絞り込みからサブフォルダーを含む検索までシームレスにつながります。Ctrl+F でフォーカス、Esc でクリア。

Ctrl+FIncrementalRecursive
◫
Views & Preview

用途に合わせた表示

詳細・一覧・なめらかにズームできるアイコン表示。画像・動画・PDF・RAW のサムネイル、Alt+P のプレビューペイン、列のカスタマイズとコンパクトビューに対応。表示と並べ替えはフォルダーごとに記憶されます。

DetailsIcons zoomThumbnailsAlt+PPer-folder
🌲
Sidebar & Bookmarks

ツリーもお気に入りも

Windows 標準のクイックアクセスに加え、ワンクリックでフォルダーツリー表示へ切り替え。現在地に自動追従します。Chrome 互換のお気に入りバーも Ctrl+Shift+B で。

◐
Theme & Icons

テーマとアイコンを選べる

Windows のダーク / ライトとアクセントカラーへ即時追従、One Dark とアクリル効果も選択可能。ファイルアイコンは絵文字 / Material / Windows 標準の 3 セットから。

⚡
Native AOT

軽快なネイティブ起動

.NET 10 Native AOT の自己完結型アプリ。別途ランタイムをインストールする必要はありません。

↻
Auto Update

いつでも最新版に

起動時と 6 時間おきに更新を自動チェック。ダイアログからダウンロード・適用・再起動まで完結し、スキップも選べます。

Design Principle

新しく覚えなくていい、
新しいファイラー。

A new file manager you already know how to use.

Browser-shaped
タブ、アドレス入力、お気に入りバーChrome の操作語彙を採用
Explorer-shaped
パンくず、クイックアクセス、ファイル操作Windows Explorer の作法を維持
Kiriha
二つの「いつもの」を、一つの流れに慣れたまま、移動だけが速くなる
01
Windows と共存する

標準 API とシェル機能を使い、Explorer との行き来を分断しません。

02
状態はタブごとに保つ

履歴、検索、並べ替え、表示モードは開いている場所ごとに独立し、フォルダーごとに記憶されます。

03
作業は次回へつなぐ

ウィンドウ位置、タブ、固定タブ、起動フォルダーを復元できます。
