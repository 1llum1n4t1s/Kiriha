タブで軽やかに、
Explorer の使い勝手そのままに。

Kiriha（キリハ）は、Chrome のような垂直タブ・お気に入りバーと、Windows Explorer そのままのファイル操作をひとつの画面にまとめた、オープンソースのファイルマネージャーです。

14 日間の無料試用 · その後は ¥980 買い切り（サブスクなし）

Features

慣れた操作に、タブの身軽さを。

Windows の作法は置き換えず、複数フォルダーを行き来する日常だけをもっと軽やかに。

🗂️

迷わず使える垂直タブ

新規タブ、固定、複製、復元、ドラッグ並べ替え、中クリックで閉じる。フォルダーをタブバーへドロップすれば新しいタブで開きます。

Ctrl+TCtrl+Tabピン留め
🖱️

本物のシェルメニュー

コピー、削除、プロパティ、Open with、拡張機能の項目まで、Windows 標準の右クリックメニューがそのまま動きます。「新規作成」も Explorer と同じ内容です。

↔️

コピーも DnD も相互互換

Explorer と Kiriha の間で切り取り・コピー・貼り付け、ドラッグ＆ドロップが自然につながります。削除はごみ箱へ、進捗も Windows 標準ダイアログ。共有シートにも対応。

🔍

探す操作をすぐ近くに

検索ボックスに入力するだけで、現在のフォルダーの絞り込みからサブフォルダーを含む検索までシームレスにつながります。

Ctrl+Fインクリメンタル再帰検索
🖼️

用途に合わせた表示

詳細・一覧・なめらかにズームできるアイコン表示に、スライダー最大で切り替わるギャラリーモード。画像・動画・PDF・RAW・JXL/AVIF/HEIC のサムネイルに対応します。

🌲

ツリーもお気に入りも

クイックアクセスと XP 風フォルダーツリーを切り替えられる左ペイン、Chrome 互換のお気に入りバー。プレビューペインで画像やテキストも即確認。

🎨

テーマとアイコンを選べる

ライト / ダーク / One Dark と Windows のアクセントカラーに追従。ファイルアイコンは絵文字・Material・Windows 標準の 3 セットから選べます。

⚡

軽快なネイティブ起動

.NET Native AOT による単一ネイティブバイナリ。ランタイム不要で起動が速く、x64 / ARM64 の両方にネイティブ対応します。

Native AOTx64ARM64
🔄

いつでも最新版に

起動時と 6 時間おきに更新を自動チェック。新しいバージョンはダイアログからダウンロード・適用・再起動まで完結します。
