# Kiriha

Chrome ライクな見た目のタブ型ファイラー（Windows 用）。

最新版は [Kiriha 公式サイト](https://kiriha.nephilim.jp/) からダウンロードできます。
インストールすると、スタートメニュー直下とデスクトップにKirihaのショートカットが作成されます。

## 主な機能

- **垂直タブ**: `＋` で新規タブ、`✕` で閉じる（最後の 1 タブを閉じても新しいタブで維持）。右クリックで **タブの固定**（固定タブはその階層を維持し、移動先は新しいタブで開く）。タブバーの幅と固定タブは次回起動時に復元
- **ナビゲーション**: 戻る / 進む / 更新 / 1 つ上へ
- **アドレスバー**: Windows 11 と同じパンくずリスト。空白クリックで現在のパスをコピーして直接入力モードへ切り替え、画面の空白クリックまたは Esc でパンくずへ戻る。Enter で移動。「PC」でドライブ一覧
- **左ペイン**: Windows 標準のクイックアクセス（ピン留めフォルダー）と PC / ドライブを表示
- **コマンドバー**: エクスプローラー同等の「新規作成」（レジストリ ShellNew 由来の項目）と「表示」（アイコンサイズ / 一覧 / 詳細、隠しファイル / 拡張子の切替）。表示種別・アイコンサイズ・並べ替えはフォルダーごとに保存
- **右クリックメニュー**: Windows 標準のシェルコンテキストメニュー（コピー / 削除 / プロパティ、拡張機能が追加した verb など）が直接動作
- **ファイル操作**: 切り取り / コピー / 貼り付け（Ctrl+X/C/V、エクスプローラーとクリップボード完全互換）、削除（ごみ箱 + 進捗ダイアログ）、名前の変更（F2）、並べ替え、プロパティ
- **ドラッグ & ドロップ**: エクスプローラーとの相互 DnD 対応。同一ドライブ = 移動 / 別ドライブ = コピー（Ctrl でコピー、Shift で移動を強制）。複数選択、フォルダー行への直接ドロップ、カーソル追従プレビューとドロップ先の強調表示に対応
- **テーマ**: Windows のダーク / ライト設定とアクセントカラーに自動追従（変更即反映）。アクリル効果のON/OFFに対応し、垂直タブバーはタイトルバーと連続した背景で表示
- **ファイル一覧**: 名前 / 更新日時 / 種類 / サイズ。フォルダーはダブルクリックで移動、ファイルは関連付けアプリで起動。小アイコンと一覧はWindows標準に合わせた配置で、アイコン表示は大量項目でも表示範囲だけを描画
- **アイコンセット**: 設定タブのドロップダウンから、従来の絵文字、[Material Icon Theme](https://github.com/material-extensions/vscode-material-icon-theme)（MIT）の拡張子別カラーアイコン、エクスプローラーと同じWindows標準Shellアイコンを選択可能（詳細 / 一覧 / アイコン表示すべてに反映）
- **ステータスバー**: エクスプローラー同等（項目数 / 選択数と合計サイズ）
- **お気に入りバー**: Ctrl+Shift+B で表示切替。DnD で登録、フォルダー作成 / 名前変更 / 並べ替えなど Chrome 互換の右クリックメニュー
- **検索**: ツールバー右の検索ボックスで現在のフォルダーを絞り込み（Ctrl+F、Esc でクリア）
- **詳細表示**: カラムヘッダークリックでソート（昇順 / 降順トグル）、境界ドラッグで幅変更
- **プレビューペイン**: Alt+P で画像 / テキスト / フォルダー情報（項目数・合計サイズ）をプレビュー
- **検索**: インクリメンタル絞り込み + Enter でサブフォルダーを含む再帰検索
- **詳細表示**: 列の表示 / 非表示（ヘッダー右クリック、作成日時列あり）、列幅変更、コンパクトビュー、ドライブ使用率バー、画像・動画・PDF・RAWサムネイル（アイコンビュー）
- **自動更新検知**: FileSystemWatcher でフォルダーの変更を自動反映
- **セッション**: ウィンドウ位置 / サイズ / 最大化 / 開いていたタブの復元（設定）、起動フォルダー設定、テーマ選択（システム / ライト / ダーク / One Dark）
- **キーボード / マウス**: Ctrl+T/W/Tab・Ctrl+1..9・Ctrl+PgUp/PgDn（タブ）、Ctrl+L/F4（アドレス）、Ctrl+F/F3（検索）、F5、Alt+←/→/↑/Home、F2、Delete / Shift+Delete、Ctrl+D、Alt+Enter、F11、Ctrl+0、マウスの戻る / 進むボタン、タブの中クリック閉じ・ドラッグ並べ替え（一覧は設定タブの「キーボードショートカット」参照）
- **オプション**: 「… > オプション」で同一ウィンドウ内に設定画面（表示設定 / 更新設定）
- **自動更新**: Velopack による自動更新（起動時チェック + 6時間おきの定期チェック + 「… > 更新の確認」で手動チェック）。新バージョンはダイアログからダウンロード / 適用 / 再起動まで完結。「このバージョンをスキップ」も可能
- **場所移動**: コマンドバーの「場所」からホーム、各ユーザーフォルダー、AppData、Program Files、Windows、一時フォルダーなどへ直接移動
- **ファイル種別フィルター**: コマンドバーの「種類」からメディア、文書、開発ファイル、アーカイブ、設計データなどを複数拡張子単位で絞り込み
- **選択・並べ替え・表示の拡張**: フォルダーのみ／ファイルのみの選択、よく使う並べ替え条件、表示モードを通常メニューから実行
- **コマンドパレット**: `Ctrl+Shift+P` または「… > コマンドパレット」からアプリ内の機能を横断検索して実行
- **追加ツール**: タブ操作20件と、コマンドバーの「ツール」から使えるファイル／フォルダー操作40件、フォルダー操作20件、条件選択20件。Windows標準コンテキストメニューは置き換えない

## ビルド・実行

```bash
dotnet run --project src/Kiriha/Kiriha.csproj
```

### ネイティブ AOT publish

```bash
dotnet publish src/Kiriha/Kiriha.csproj -c Release
```

出力: `src/Kiriha/bin/Release/win-x64/publish/Kiriha.exe`

> Git Bash からローカル publish する場合、AOT リンク段の vswhere 解決のため
> `C:\Program Files (x86)\Microsoft Visual Studio\Installer` を PATH に追加してください。
