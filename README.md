# Kiriha

Chrome ライクな見た目のタブ型ファイラー（Windows 用）。エクスプローラーと同じ操作感のまま、垂直タブ・お気に入りバー・コマンドパレットで快適にファイル管理できます。

## インストール

1. [Kiriha 公式サイト](https://kiriha.nephilim.jp/) からインストーラーをダウンロードして実行します。
2. スタートメニューとデスクトップに Kiriha のショートカットが作成されます。
3. 以後の更新は自動です（起動時と 6 時間おきにチェック、「… > 更新の確認」で手動チェックも可能）。新バージョンはダイアログからダウンロード・適用・再起動まで完結し、「このバージョンをスキップ」も選べます。

## 主な機能

### タブとナビゲーション

- **垂直タブ**: `＋` で新規タブ、`✕` で閉じる。右クリックから複製・固定・一括クローズなどのタブ操作。固定タブはその階層を維持し、移動先は新しいタブで開きます。タブはドラッグで並べ替えでき、フォルダーをタブバーへドロップすると新しいタブで開きます
- **アドレスバー**: Windows 11 と同じパンくずリスト。空白クリックでパス直接入力へ切り替え（Esc で戻る）、「PC」でドライブ一覧
- **左ペイン**: Windows 標準のクイックアクセスと PC / ドライブを表示。「ツリー表示」トグルでフォルダーツリーに切り替わり、現在のフォルダーへ自動で追従します。ツールバーのボタンで左ペイン自体の表示 / 非表示も可能
- **お気に入りバー**: Ctrl+Shift+B で表示切替。ドラッグ & ドロップで登録、フォルダー分け・名前変更・並べ替えなど Chrome と同じ右クリックメニュー

### ファイル操作

- **エクスプローラー完全互換**: 切り取り / コピー / 貼り付け（Ctrl+X/C/V、エクスプローラーとクリップボード相互運用）、削除はごみ箱へ（進捗ダイアログ付き）、F2 で名前の変更
- **右クリックメニュー**: Windows 標準のシェルコンテキストメニューがそのまま動作（コピー / 削除 / プロパティ、拡張機能が追加した項目も含む）
- **ドラッグ & ドロップ**: エクスプローラーとの相互 DnD。同一ドライブ = 移動 / 別ドライブ = コピー（Ctrl でコピー、Shift で移動を強制）。ドロップ先はプレビューで確認できます
- **新規作成**: エクスプローラーと同じ「新規作成」メニュー（インストール済みアプリの項目も表示）
- **自動更新**: フォルダー内の変更（追加・削除・名前変更）は自動で一覧へ反映されます

### 表示とカスタマイズ

- **表示モード**: 詳細 / 一覧 / アイコン（特大〜小、なめらかにサイズ変更可能）。表示種別・アイコンサイズ・並べ替えはフォルダーごとに記憶されます
- **詳細表示**: 列ヘッダークリックでソート、右クリックで列の表示 / 非表示（作成日時列あり）、境界ドラッグで幅変更。コンパクトビュー（全タブ共通）にも対応
- **サムネイル**: アイコン表示で画像・動画・PDF・RAW のサムネイルを表示。ドライブには使用率バーを表示
- **アイコンセット**: 設定から 3 種類を選択 — 絵文字 / [Material Icon Theme](https://github.com/material-extensions/vscode-material-icon-theme) の拡張子別カラーアイコン / エクスプローラーと同じ Windows 標準アイコン
- **テーマ**: Windows のダーク / ライト設定とアクセントカラーに自動追従（変更即反映）。システム / ライト / ダーク / One Dark から選択でき、アクリル効果の ON/OFF も可能
- **プレビューペイン**: Alt+P で画像 / テキスト / フォルダー情報（項目数・合計サイズ）をプレビュー
- **ステータスバー**: 項目数 / 選択数と合計サイズ / ドライブの空き領域を表示

### 検索とフィルター

- **検索**: 検索ボックスに入力するだけでサブフォルダーを含めて検索（Ctrl+F でフォーカス、Esc でクリア）
- **種類フィルター**: コマンドバーの「種類」からメディア・文書・開発ファイル・アーカイブなどを拡張子グループ単位で絞り込み
- **条件選択**: フォルダーのみ / ファイルのみ / 画像のみ、などの一括選択

### コマンドパレットとツール

- **コマンドパレット**: Ctrl+Shift+P でアプリ内の全機能を横断検索して実行
- **ツールメニュー**: コマンドバーの「ツール」からターミナルで開く・パスのコピー・ハッシュ計算などファイル / フォルダー操作を多数実行

## 設定

「… > オプション」または左ペイン下部の「設定」から、同一ウィンドウ内の設定タブが開きます。

- 起動時に開くフォルダー / 前回のタブを復元
- テーマ・アクリル効果・アイコンセット
- 隠しファイル / 拡張子 / チェックボックスの表示
- キーボードショートカット一覧の確認
- 更新の確認とバージョン情報

ウィンドウ位置・サイズ・タブ・お気に入り・表示設定は自動で保存されます。

## 主なキーボードショートカット

| 操作 | キー |
|---|---|
| 新しいタブ / タブを閉じる | Ctrl+T / Ctrl+W |
| タブの切り替え | Ctrl+Tab、Ctrl+1〜9、Ctrl+PgUp/PgDn |
| アドレスバーへ移動 | Ctrl+L または F4 |
| 検索 | Ctrl+F または F3 |
| 戻る / 進む / 1 つ上へ | Alt+← / Alt+→ / Alt+↑ |
| 更新 | F5 |
| 名前の変更 / 削除 / 完全削除 | F2 / Delete / Shift+Delete |
| お気に入りに追加 / お気に入りバー表示 | Ctrl+D / Ctrl+Shift+B |
| プロパティ / プレビューペイン | Alt+Enter / Alt+P |
| コマンドパレット | Ctrl+Shift+P |
| 全画面 / アイコンサイズのリセット | F11 / Ctrl+0 |

全一覧は設定タブの「キーボードショートカット」で確認できます。マウスの戻る / 進むボタン、タブの中クリック閉じにも対応しています。

## トラブルシューティング

- **更新が来ない**: 「… > 更新の確認」で手動チェックできます。プロキシ環境などでダウンロードに失敗する場合は、[公式サイト](https://kiriha.nephilim.jp/) から最新のインストーラーを直接実行しても設定は引き継がれます
- **表示がおかしい / 動作が不安定**: 設定タブの「設定をリセット」で表示設定を初期状態に戻せます
- **問題の報告**: ログは `%LocalAppData%\Kiriha\logs` に保存されています。不具合報告の際に添付いただくと調査がスムーズです
